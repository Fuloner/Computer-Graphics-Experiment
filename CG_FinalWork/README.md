### 项目选题描述
基于3D位置的流体模拟
3D流体模拟听起来可能是一项极其复杂的数学挑战，但请不用担心。在图形学中，除了使用网格，我们还可以将水看作是由成千上万个微小的“小水珠（球形粒子）”组成的。
本项目基于经典的Position Based Fluids (PBF) 算法。与传统的受力分析不同，PBF直接通过迭代修正粒子的位置来强行满足“水不可压缩（密度恒定）”的物理约束。这使得模拟过程极其稳定，不会因为时间步长过大而导致系统爆炸。从一滩水花到汹涌的波浪，都将由这些神奇的粒子演绎。
请在一个3D的包围盒（如一个透明的水箱）内，实现PBF算法并渲染出流体效果。
